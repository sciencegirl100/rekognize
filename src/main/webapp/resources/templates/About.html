Rekognize

Rekognize is a web application that allows you to recognize faces in a picture. Using AWS Rekognition as the brains behind the operation, it can find faces and recognize properties of each.

It is a proof of concept and is not intended to be used in production.

Get Started

Start by heading over to the Configure tab to enter your Amazon Web Services credentials. Doing so will unlock the Run tab to use Rekognition.
